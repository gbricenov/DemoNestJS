## Description

Microservicio de Backend de prueba en Nest. Incorporta un módulo de Health-Check y Api-Auto.

## Installation

```bash
$ npm install
```

## Ambiente

* APP_PORT: puerto por defecto.
* API_PRUEBA_OBTENER_AUTOS_POR_CLIENTE: servicio para consultar autos por cliente.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# watch mode en mac/linux
$ npm run start:dev:mac

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
